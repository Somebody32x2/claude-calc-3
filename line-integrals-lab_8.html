Vector fields & line integrals

Until now f gave a number at every point. A vector field gives an arrow — a wind, a current, a force. The question changes to match: not "how high is it here?" but "how much does this field help me as I travel along a path?" Add that up along the route and you get work — and the startling fact that for some fields the route doesn't matter at all.

F(x, y) = ⟨P, Q⟩
An arrow at every point

Two ordinary functions bundled together: P is the arrow's x-component, Q its y-component. Wind maps, electric fields and fluid flows are all exactly this.

∫C F · dr
Add up the helpful part

At each step, only the component of F along your direction counts — a dot product again. Sum those contributions over the whole path and you have the work.

F = ∇f ⟹ path-independent
Some fields forget the route

If the field is the gradient of some f, the integral collapses to f(end) − f(start). The whole journey in between cancels out.

01

Walking through a field

Pick a field and a route from (0,0) to (1,1). Slide along it and watch the coral field arrow, the violet direction arrow, and the running total of their dot product.

To turn "add up the helpful part" into arithmetic, parametrize the path: write it as r(t) = ⟨x(t), y(t)⟩ for t from a to b. Then dr = r′(t) dt, and everything becomes a single ordinary integral in t:

∫C F · dr = ∫ab F(r(t)) · r′(t) dt

That is the whole computational recipe, and it is worth saying plainly what each piece does. F(r(t)) means substitute the path into the field — wherever the field formula says x, write x(t). r′(t) is the velocity along the path. Their dot product is a single number for each t, and integrating it is Calc 1. The panel below plots that dot product as t runs, so the work really is the signed area under a curve.

02

Does the route matter?

Same field, same endpoints, four different routes. For some fields every route gives the same number; for others they disagree wildly. That split is the most important idea in this lab.

Run the table below. When all four routes agree, the field is conservative: it has a potential function f with F = ∇f, and the integral is just f(end) − f(start) — the fundamental theorem of calculus, promoted to curves. When the routes disagree, no such f can exist, because a potential would have forced them to agree.

There is a cheap test that avoids trying all routes. If F = ∇f = ⟨fx, fy⟩, then P = fx and Q = fy, so ∂P/∂y = fxy and ∂Q/∂x = fyx. Mixed partials of a nice function are equal, so:

conservative ⟹ ∂Q/∂x − ∂P/∂y = 0

That quantity is the two-dimensional curl, and it measures how much the field spins. On a region with no holes the implication runs both ways, so a zero curl is a reliable green light.

03

Worked by hand, with the numbers

The mechanics, done slowly and completely. No steps skipped, every substitution written out — this is exactly what you would put on paper.

Compute ∫C F · dr for F = ⟨−y, x⟩ along the parabola y = x² from (0,0) to (1,1)

The path-dependent case. Watch how the answer depends on the route we chose.

  1. Parametrize the path. The curve is y = x², so let x = t and y = t². At t = 0 we're at (0,0); at t = 1 we're at (1,1). So t runs 0 → 1.
    r(t) = ⟨t, t²⟩,   0 ≤ t ≤ 1
  2. Differentiate to get the velocity. Differentiate each component separately.
    r′(t) = ⟨1, 2t⟩
  3. Substitute the path into the field. F = ⟨−y, x⟩, and on our path y = t², x = t.
    F(r(t)) = ⟨−t², t⟩
  4. Take the dot product. Multiply matching components and add.
    F · r′ = (−t²)(1) + (t)(2t)
            = −t² + 2t²
            = t²
  5. Integrate over t. A plain Calc 1 integral now.
    ∫₀¹ t² dt = [ t³/3 ]₀¹ = 1/3 − 0
    = 1/3 ≈ 0.333
  6. Sanity-check against a different route. Along the straight line r(t) = ⟨t, t⟩: r′ = ⟨1,1⟩, F = ⟨−t, t⟩, dot = −t + t = 0, so the work is 0. Different route, different answer — this field is not conservative, and the table in section 02 shows the same thing.

Compute ∫C F · dr for F = ⟨2xy, x²⟩ — the easy way and the hard way

The conservative case, done twice: once by brute force, once by finding a potential. Same answer, wildly different effort.

  1. Test whether it's conservative first. Here P = 2xy and Q = x².
    ∂P/∂y = ∂(2xy)/∂y = 2x   (treat x as a constant)
    ∂Q/∂x = ∂(x²)/∂x = 2x
    curl = 2x − 2x = 0  ✓ conservative
  2. Find the potential f. We need fx = 2xy. Integrate with respect to x, treating y as constant:
    f = ∫ 2xy dx = x²y + g(y)
    The constant of integration can be any function of y alone — call it g(y).
  3. Pin down g(y) using the other component. Differentiate our f with respect to y and match it to Q:
    fy = ∂(x²y + g(y))/∂y = x² + g′(y)
    must equal Q = x²
    ⟹ g′(y) = 0 ⟹ g = constant
    so  f(x,y) = x²y
  4. Apply the fundamental theorem. No parametrizing, no dot products — just evaluate at the two endpoints.
    ∫C F · dr = f(1,1) − f(0,0)
               = (1)²(1) − (0)²(0)
               = 1 − 0
    = 1
  5. The hard way, for comparison. Along r(t) = ⟨t, t²⟩: r′ = ⟨1, 2t⟩, and F(r(t)) = ⟨2·t·t², t²⟩ = ⟨2t³, t²⟩.
    F · r′ = (2t³)(1) + (t²)(2t) = 2t³ + 2t³ = 4t³
    ∫₀¹ 4t³ dt = [ t⁴ ]₀¹ = 1  ✓ same answer
    Both routes work; the potential method took a fraction of the effort and would have given the same 1 for any path between those points.
04

Common misconceptions

✕ the myth
"Zero curl means the field is conservative. Always."
✓ actually

Only on a region with no holes (simply connected). The standard counterexample is the vortex F = ⟨−y/(x²+y²), x/(x²+y²)⟩: its curl is zero everywhere it is defined, yet its circulation around the unit circle is 2π, not 0 — so it cannot be a gradient. The catch is the puncture at the origin. Zero curl gives you conservative locally; you need the domain to have no holes to upgrade that to globally. This is the one hypothesis worth actually remembering.

✕ the myth
"A line integral depends on how fast you traverse the path."
✓ actually

It does not. Reparametrize r(t) = ⟨t,t⟩ as ⟨t², t²⟩ and both F and r′ change — but the extra speed in r′ is exactly cancelled by the shorter time spent there. What does matter is the direction: reversing the path negates the integral, because every r′ flips sign.

✕ the myth
"∫ F · dr is the length of the path, weighted by F."
✓ actually

Only the component of F along the path counts, and it can be negative. Move perpendicular to the field and you accumulate exactly zero, no matter how far you travel — same geometry as the directional derivative in Lab 03. A field pushing against you subtracts. That is why work can come out negative, or zero on a long journey.

✕ the myth
"The potential f is unique."
✓ actually

It is unique only up to an additive constant — x²y and x²y + 7 are both potentials for ⟨2xy, x²⟩. This never matters for line integrals, because the constant cancels in f(end) − f(start). It is the same freedom as the "+C" in a one-variable antiderivative.

✕ the myth
"When integrating F(r(t)), plug the path into only one of the two slots."
✓ actually

Every x and y in both components of F must be replaced by x(t) and y(t). A half-substituted integrand still contains x or y after you have committed to t, and the integral becomes meaningless. Check before integrating: the expression should contain nothing but t.
